Paint App in Android

PUBLISHED ON SEP 6, 2015

Recently I was learning about Canvas and Custom views in Android and to put the theory in practice I made a pretty simple **Paint App. **

Source code of the app can be found https://github.com/priyankvex/Paint-App-Android.

[caption id=“attachment_159” align=“aligncenter” width=“480”] Screenshot of the app[/caption]

The App is pretty simple with features 1. Create Drawings (Obvious one :P). 2. Select different brush sizes. 3. Eraser with different sizes. 4. Color pallet. 5. Save drawings in Gallery.

Source code of the app can be found https://github.com/priyankvex/Paint-App-Android.
