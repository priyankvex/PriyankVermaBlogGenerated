A Basic Calculator using Swing and JAVA(TM).

Thu, May 22, 2014

Hi,there . This time i would like to share my new work. This a basic four function calculator made using java and Swing. The GUI created is with the help of….you guessed it right SWING.

Java simple calculator

  1. A four-function calculator with the following functions:
  2. Addition – adds two numbers
  3. Subtraction – subtracts number two from number one
  4. Multiplication – Multiplies two numbers

  5. Division – divides number two into number one

  6. Used JButton for the numbers and functions.

  7. Created a class to respond to the events caused by numbers, functions, exit, help and about functionality.

The run-able .jar file can be downloaded freely from the link. https://www.dropbox.com/s/x4g6okw78248pqe/calculator.jar

Any comments or bug reports are more then welcome.
