Starting c/c++ programming in Linux.

PUBLISHED ON MAR 11, 2014

So, guys this post is just a kick off to my blog.

This post is about coding in c or c++ in Linux . So, if you are a beginner looking to get your hand on coding in c or c++ in Linux, this will be of great help.

STEPS:

1.You can use gedit or i recommend you download Sublime Text for Linux.

  1. Select your language as c/c++ and code and save the file.

  2. Open the terminal and use cd (change directory ) command to get to the directory in which you have saved your program .

  3. Now, type “ gcc -o name name.c” and hit enter. This will create the object code of your program and compile it and reports the error( if any ) which can be read by your computer,

5.To run your program ,type “./name”. And hit enter.

  1. And you are done.
